## ⚠️ Open validation
**Reddit was hard-blocked during research** — persona detail leans on Indie Hackers, HN, and Substack. The behavioral shape is well-sourced; an authenticated Reddit pass is the #1 follow-up to confirm trigger intensity, reply-safety panic, scrolling-waste, and "would you pay." Setup options (Reddit OAuth script helper, or a Reddit MCP server) are scoped and ready to wire up when credentials are added (`research/.reddit.env`, gitignored). Findings dependent on this layer are tagged "⚠️ Pending Reddit validation" in the phase files.

## Headline conclusions
1. **Creator-primary.** Voice/identity layer is the product; agent/MCP is a distribution channel.
2. **Wedge = the closed own-analytics voice loop** — must be *demonstrated*, not claimed (buyers can't tell it from $9.99 style-mimicry on a feature list).
3. **Primary ICP = the Build-in-Public Founder/Operator;** secondary = Scaling Ghostwriter/Agency (build a proof-gated multi-account tier); channel/future = Creator-Who-Automates.
4. **Riskiest thing = activation:** the first "yes, this sounds like me" moment. North-star = time-to-first-sounds-like-me.
5. **Biggest build lever = make voice depth visible** (the Voice Report).
